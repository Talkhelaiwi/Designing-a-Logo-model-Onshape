# SSC Napoli Logo – Onshape

## Project Overview

This project recreates the official SSC Napoli logo as a 3D CAD model using Onshape. The design was created by tracing a reference image, creating sketches, and using extrusion features to build the final 3D model. The completed design was exported as an STL file, making it ready for 3D printing.

---

## Software Used

- Onshape

---

## Design Process

1. Looking for an image of the SSC Napoli logo to Tske as an example.
2. Created the circular outline using sketches.
3. Added the "N" letter using the Text tool.
4. Used Extrude features to create the 3D model.
5. Applied colors similar to the original logo.
6. Exported the completed model as an STL file.

---

## Dimensions

- Base thickness: 4 mm
- Letter thickness: 2 mm
