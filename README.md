# SSC Napoli Logo – Onshape

## Project Overview

This project is a 3D model of the SSC Napoli logo created using Onshape.

The design was made by using a reference image of the original logo, drawing the circular shapes, adding the letter N, and applying extrusion features to turn the sketch into a 3D model.

The final design was exported as an STL file and can be used for 3D printing.

## Final Onshape Design

This is the final 3D model created in Onshape.

The model includes:

- A circular outer frame
- A circular inner area
- The letter N in the center
- A base thickness of 4 mm
- A raised letter thickness of 2 mm

![Final Onshape Model](screenshots/onshape-model.jpg)

---

## Design Process

The model was created through the following steps:

1. A reference image of the SSC Napoli logo was selected.
2. Two circles were drawn to create the outer frame and the inner area.
3. The letter N was added using the Text tool.
4. The main circular shape was extruded by 4 mm.
5. The letter N was extruded by 2 mm.
6. Colors were applied to make the model look similar to the original logo.
7. The completed model was exported as an STL file.
